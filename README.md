
##  Text UI

The best Text UI to replace the default 3DText!

##  🌟 Feature List 
- ⌨️ Key support
- 👥 Compatibility with (luke_textui and cd_drawtextui)
- 🖥️ Multiple UI's
- 🔊 Popup sound
- ⬆️ Multiple alignment possibilities (top, bottom, right, left)
- ➡️ Works on all resolutions
- 🚀 Standalone script
